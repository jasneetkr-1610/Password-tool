# Password Tool (Python)

## Description
A menu-driven Python CLI tool to check password strength and generate strong passwords with an option to include user initials.

## Features
- Password strength checker
- Strong password generator
- Option to include user initials
- Menu-driven interface

## Concepts Used
- Functions
- Loops
- Conditional statements
- String methods
- Random module

python main.py
